## Experiment No: 1d – Conditional Statements- Checking Vowel or not

## AIM  
To Write a Python program to check whether the given character is a vowel or not using if..else statement
## ALGORITHM  
1. Begin the program.  
2. Read a character input from the user and store it in the variable char.
3. Check if char is present in the string 'AEIOU'.
4. If it is a vowel, display "The given character is a vowel"
5. Otherwise, display "The given character is NOT a vowel"
6. Terminate the program.

## PROGRAM
```python
# Reg.No-212223090030
# Name-Vijayalakshmi M P
char=input()
if char in 'AEIOU':
    print("The given character is a vowel")
else:
    print("The given character is NOT a vowel")
```
